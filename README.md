# AI Engineering Syllabus

This repository contains the **AI Engineering syllabus** and related learning materials for 4Geeks Academy:

- **Curriculum content** (milestones, skills, contexts and theory).
- **Hands-on projects** for each skill and milestone, under `content/projects/`.
- Supporting files for the platform (`learn.json`, reference solutions, assets, etc.).

---

## Projects (hands-on practice)

You can find the practical projects in [`content/projects`](./content/projects):

1. **[Artist landing: HTML, CSS, SEO and accessibility](./content/projects/html-css-artist-landing-seo-access)**  
   Accessible, SEO-optimized landing page for an artist using semantic HTML and CSS.

2. **[Simple dashboard with Tailwind CSS](./content/projects/simple-dashboard-tailwind-css)**  
   Responsive dashboard with HTML and Tailwind showing KPIs, drivers, and operational details (no React).

3. **[Company Project Milestone: Web Fundamentals](./content/projects/ai-eng-milestone-web-fundamentals)**  
   Milestone 1 — Company public website: landing page plus application/sign-up form with semantic HTML5, Tailwind, Schema.org, and JavaScript validation.

4. **[Collaborative project: online store with HTML and Tailwind](./content/projects/collaborative-project-html-tailwind-online-store)**  
   Collaborative e-commerce prototype (Home, Catalog, Product, Cart, Checkout) with HTML and Tailwind, using Git branches and pull requests.

5. **[Cinema Seat Manager (TypeScript)](./content/projects/seats-management-typescript)**  
   CLI cinema seat reservation system using a 2D array (reserve, count, adjacent-seat search).

6. **[Data modeling and class diagrams](./content/projects/data-modeling-and-class-diagrams)**  
   UML-style class diagrams for a playlist player and a digital wallet, focused on entities, data types, and relationships.

7. **[Programming Fundamentals Milestone (TypeScript)](./content/projects/ai-eng-milestone-coding-fundamentals)**  
   Milestone 2 — Programming fundamentals with TypeScript: small, testable modules for control flow, arrays, objects, functions and edge cases.

8. **[AI Agent Rental Platform: Admin panel prototype](./content/projects/agent-hub-ui-specs-and-prompts)**  
   Spec-driven frontend prototype for a multi-view admin panel built with HTML, Tailwind, and vanilla JavaScript interactions.

9. **[Talk to the Machine: Chat interface with a real AI API](./content/projects/chat-interface-real-ai-api)**  
   Browser chat interface that calls the Groq API with `fetch`, sends full conversation history, and tracks token usage plus response metrics.

10. **[Voice to-do list with AI API](./content/projects/voice-to-do-list-api)**  
    Voice-powered to-do workflow that captures user input and uses an AI API to transform spoken requests into actionable tasks.

For more details on each project, open its folder inside `content/projects/` and read the corresponding `README.md` (and `README.es.md` when available).
