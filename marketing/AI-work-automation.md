## Module 01 ยท How AI Really Works

**Week 1 ยท Self-paced + Live Class**

`ChatGPT` `Claude` `Prompting`

In this first module, students build the mental model that makes everything else click. They learn what LLMs are, how they work in everyday language, and why choosing the right model for the task affects cost, speed, and output quality. The distinction between a chatbot, an assistant, and an agent is established โ€” concepts that will be fundamental for the rest of the program.

The hands-on part starts with a work audit exercise: the student lists their 10 most repetitive weekly tasks, identifies their top 3 automation candidates, and writes their first 5 real prompts applied to their job. The live class includes Q&A and peer sharing on which tasks they are automating.

**Skills developed:**

- Explain how LLMs work without technical jargon
- Choose the right AI model for the type of task
- Distinguish between chatbot, assistant, and agent
- Run an audit of repetitive tasks in their own work
- Write basic prompts with role, context, format, and constraints

## Module 02 ยท Prompting as a Skill

**Week 2 ยท Self-paced + Live Class**

`ChatGPT` `Claude` `Prompting` `Markdown` `Notion` `Google Docs`

This module treats prompting as a professional skill to practice and refine. Students learn the most effective prompting patterns: chain of thought, few-shot, persona, and structured output. They learn to generate emails, reports, meeting summaries, and research reliably and repeatably, controlling output format with Markdown.

A key component is context management: what the model remembers, what it forgets, and how to structure conversations for consistent results. Students finish the module with a personal prompt library in Notion or Google Docs โ€” a real, reusable professional asset.

**Skills developed:**

- Apply advanced prompting patterns (chain of thought, few-shot, persona, structured output)
- Generate emails, reports, and research with precise, reusable prompts
- Control output format using Markdown
- Manage AI conversation context effectively
- Build and maintain a personal prompt library

## Module 04 ยท Automation Pipelines with n8n

**Week 4 ยท Self-paced + Live Class ยท Final Project**

**Skills developed:**
