### 01 · 💬 Prompt Engineering
`ChatGPT` `Claude` `Notion` `Google Docs` `Markdown`

*Self-paced*

This course treats prompting as a professional skill — something you practice, refine, and systematize, not just improvise. Students learn the most effective prompting patterns used by AI power users: chain of thought, few-shot examples, persona framing, and structured output. The goal is not just getting answers from AI, but getting the right answer in the right format, reliably and repeatably.

**What you will build**

A personal prompt library in Notion or Google Docs — a real, reusable professional asset organized by use case (emails, reports, research, summaries). By the end of the course, every student has a working toolkit they can open and use the next day at work.

**Skills developed**

- Apply advanced prompting patterns: chain of thought, few-shot, persona, structured output
- Generate emails, reports, and research with precise, reusable prompts
- Control AI output format using Markdown
- Manage conversation context for consistent multi-turn results
- Build and maintain a personal prompt library organized by use case
- Distinguish when to use different models (GPT-4o, Claude, Gemini) for different tasks

### 02 · 🧠 Applied AI
`ChatGPT` `Claude` `Make` `Google Workspace` `Canva` `Perplexity`

*Self-paced*

Applied AI focuses on one thing: using AI to do real professional work faster and better. Students work through the most common use cases for knowledge workers — writing and editing, research and summarization, meeting notes, presentations, and content creation — and learn how to integrate AI into their existing workflow without changing every tool they already use.

**What you will build**

A personal AI Audit: students map out their 10 most time-consuming weekly tasks, identify which ones AI can handle, and implement AI assistance for at least 3 of them by the end of the course. The final deliverable is a documented AI workflow the student can share with their team or manager.

**Skills developed**

- Use AI for professional writing: emails, reports, proposals, and summaries
- Conduct AI-assisted research with tools like Perplexity and ChatGPT with web access
- Generate and refine presentations and visual content with AI
- Automate meeting notes and action item extraction
- Build a documented personal AI workflow for recurring work tasks
- Evaluate AI output critically and apply editing and verification practices

### 03 · ⚙️ No-Code & Automations
`n8n` `Make` `Zapier` `Gmail` `Slack` `Google Sheets` `Webhooks`

*Self-paced*

This course introduces visual workflow builders — tools that let you connect apps and automate tasks without writing code. Students learn the core concepts behind automation (triggers, actions, conditions, loops) and apply them by building real automations across Gmail, Slack, Google Sheets, and external services via webhooks. The course focuses on automations that run on their own after they are set up — no manual triggering required.

**What you will build**

At least 3 automations: one for personal productivity (e.g., auto-labeling emails), one for team collaboration (e.g., Slack notifications from a spreadsheet), and one end-to-end pipeline that includes an AI step. The final deliverable is a working automation the student keeps and actively uses.

**Skills developed**

- Build visual workflows using triggers, actions, conditions, and loops in n8n and Make
- Connect Gmail, Slack, Google Sheets, and external services via webhooks
- Design automations that run without manual intervention
- Integrate an AI model as a decision or generation step inside a pipeline
- Debug and fix broken workflows using execution logs
- Identify which tasks in a workflow are automation candidates

### 04 · ✨ Vibe Coding with AI
`Cursor` `GitHub Copilot` `ChatGPT` `Replit` `HTML / CSS / JavaScript`

*Self-paced*

Vibe Coding is the practice of building real software projects by collaborating with AI — writing natural language instructions, reviewing AI-generated code, iterating on the output, and shipping something that works. No prior programming experience is required. Students learn just enough code literacy to direct the AI, review its output, and catch errors — not to write everything from scratch.

**What you will build**

A working web project built from scratch using AI coding assistants as the primary tool: a personal portfolio page, a simple app, or a productivity tool of the student's choosing. The emphasis is on the product — something functional, deployed, and real — rather than on code syntax memorization.

**Skills developed**

- Use AI coding assistants (Cursor, Copilot) to generate and iterate on code
- Read and understand AI-generated HTML, CSS, and JavaScript at a functional level
- Describe software requirements in natural language and translate them into working code
- Deploy a web project using Replit or a similar platform
- Debug issues collaboratively with AI and identify when to verify output manually
- Maintain a personal project repository as a portfolio piece

### 05 · 🤖 OpenClaw
`OpenClaw` `JSON` `Google Drive` `Gmail` `Notion` `Slack`

*Self-paced*

OpenClaw is a personal AI assistant framework that lets students configure and deploy AI-powered skills for on-demand tasks — the kind they trigger manually when they need them, as opposed to automated pipelines. This course covers installation, configuration, and customization of OpenClaw, with a focus on connecting it to the tools students already use every day.

**What you will build**

A custom OpenClaw skill built from scratch using a prompt template — a specialized command the student's assistant can run on demand. Students also connect OpenClaw to Google Drive, Gmail, and Notion, turning it into an active layer over their existing workflow rather than a separate tool to switch between.

**Skills developed**

- Install and configure OpenClaw as a personal AI assistant
- Browse and run community-built skills for common tasks
- Read and edit JSON configuration files in context
- Create a custom skill from a prompt template
- Connect OpenClaw to Google Drive, Gmail, and Notion
- Integrate OpenClaw into an existing daily workflow

### 06 · 🐍 Backend Dev with Python
`Python` `FastAPI` `PostgreSQL` `REST APIs` `Postman` `Docker`

*Self-paced*

This course teaches students how to build backend services and APIs using Python and FastAPI — the same stack used in production AI applications. Students learn how APIs work, how to design and expose endpoints, how to connect to databases, and how to integrate an AI model (via the OpenAI or Anthropic API) as a functional component of a backend service.

**What you will build**

A deployed API with at least 3 endpoints, a connected database, and one AI-powered route — for example, a summarization endpoint, a classification step, or a chatbot handler. Students leave with a GitHub repository they can show as a portfolio piece.

**Skills developed**

- Build and deploy REST APIs using FastAPI and Python
- Connect a backend service to a PostgreSQL database
- Design clean API routes with request validation and error handling
- Integrate the OpenAI or Anthropic API as a service inside a backend
- Test API endpoints with Postman
- Containerize a Python service using Docker basics

### 07 · 📘 Deep Dive into Python
`Python` `Jupyter Notebooks` `VS Code` `pip` `Git`

*Self-paced*

Deep Dive into Python is a comprehensive course covering the Python language from foundations to advanced features. It is the recommended prerequisite or companion for Backend Dev with Python and Data Science & ML. Students who already write basic Python will close their gaps; students who are newer to programming will build a solid foundation they can apply to any technical course in the 4Geeks catalog.

**What you will build**

A series of progressively challenging exercises in Jupyter Notebooks, ending with a Python script that solves a real problem in the student's professional domain — whether that is data processing, file management, API interaction, or text manipulation.

**Skills developed**

- Write clean, idiomatic Python using proper syntax, conventions, and best practices
- Work with Python data structures: lists, dicts, sets, tuples, and comprehensions
- Apply object-oriented programming: classes, inheritance, and encapsulation
- Handle files, exceptions, and external libraries using pip
- Write and organize multi-file Python projects
- Use Git for version control and project management

### 08 · 📊 Data Science & ML
`Python` `NumPy` `Pandas` `scikit-learn` `Matplotlib` `Jupyter`

*Self-paced*

This course introduces the core toolkit of data science and machine learning using Python. Students learn to load, clean, explore, and visualize datasets, and then build and evaluate machine learning models using scikit-learn. The course is practical and project-driven — every concept is applied to a real dataset, and students graduate with two complete data projects they can present to employers or clients.

**What you will build**

Two projects: a data exploration report (loading a real dataset, cleaning it, visualizing trends, and summarizing findings) and a predictive model (training a classification or regression model, evaluating its accuracy, and interpreting its predictions). Both are built in Jupyter Notebooks and submitted as portfolio pieces.

**Skills developed**

- Load, clean, and explore datasets using NumPy and Pandas
- Visualize data and identify patterns using Matplotlib and Seaborn
- Build and train classification and regression models with scikit-learn
- Evaluate model performance using accuracy, precision, recall, and RMSE
- Interpret model output and communicate findings to non-technical audiences
- Document and present data projects in Jupyter Notebooks

## How You Graduate

AI Flex students earn a certificate for each course they complete. There is no single final project — each course has its own deliverable, and each certificate reflects a specific, demonstrable skill.

Certificates earned in AI Flex are the same certificates awarded in AI Fluency and the full AI Engineering Bootcamp. They are stackable: a student who completes AI Flex and later enrolls in the AI Engineering Bootcamp receives credit for completed courses.

| | |
|---|---|
| **Certificates** | 1 per completed course (up to 4 total) |
| **Deliverables** | Each course has a final project or portfolio piece |
| **Stackable** | Credits transfer to AI Fluency and AI Engineering Bootcamp |
| **Next step** | Students who want to go deeper can continue in AI Engineering, where the same tools and foundations extend into building full AI-powered applications |
